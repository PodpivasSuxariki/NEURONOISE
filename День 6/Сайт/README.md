# NEURONOISE — статичный сайт

Готовая адаптивная верстка по макету: главная страница, страница урока и контакты.

## Структура

- `index.html` — главная страница
- `lesson.html` — страница урока
- `contacts.html` — страница контактов
- `css/style.css` — стили, адаптив, Flexbox/Grid
- `js/main.js` — мобильное меню и демонстрационная отправка формы
- `assets/` — локальные графические ассеты

## Как запустить через OpenServer

1. Установите и запустите OpenServer.
2. Создайте папку проекта, например: `OpenServer/domains/myproject.local`.
3. Скопируйте туда все файлы из этой папки.
4. Перезапустите OpenServer.
5. Откройте в браузере `http://myproject.local/`.

## Адаптивность

Сайт адаптирован под ПК, планшеты и смартфоны. Основные сетки сделаны через CSS Grid и Flexbox.
